<h1> Informed-Searches </h1>
This repository visually shows how the listed informed searches work.
<ul>
  <li>Greedy Best First Search</li>
  <li>A*</li>
</ul>
<p>Following Libraries are a requirement:</p>
<ul>
  <li>pygame</li>
  <li>sys</li>
  <li>math</li>
  <li>heapq</li>
  <li>random</li>
  <li>time</li>
</ul>
<pre><h2>Dynamic Pathfinding Agent</h2>
HOW TO USE:
  - Draw walls  : left-click and drag on the grid
  - Erase walls : right-click and drag on the grid
  - Use the panel on the right to control everything
KEYBOARD SHORTCUTS:
  SPACE   run search
  R       random maze
  C       clear grid
  D       toggle dynamic obstacles
  1       switch to A*
  2       switch to GBFS
  M       Manhattan heuristic
  E       Euclidean heuristic
  S       place-start mode (then click a grid cell)
  G       place-goal  mode (then click a grid cell)
  ESC     cancel current action </pre>
